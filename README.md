# bloGO

## MAINLY BACKEND PRACTICE

### Start
Clone or Download Code zip

### Backend 
- SpringBoot
- cd to project directory
- cd backend
- 1 Run: `mvnw spring-boot:run`

### Frontend
- React 
- cd to project directory
- cd frontend
- 1 Run: `npm install` or `yarn install`
- 2 Run: `npm run start` 
- Redirect  to (http://localhost:3000)
